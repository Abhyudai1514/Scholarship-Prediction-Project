# 🎓 Scholarship Eligibility Prediction API

## 📌 Problem Statement
Educational institutions process large volumes of student applications, making scholarship eligibility evaluation time-consuming and inconsistent. This project aims to automate and improve the decision-making process using predictive analytics.

---

## 📊 Dataset
The dataset contains student academic and demographic attributes such as:

- Exam scores  
- Department  
- Country  
- Academic performance indicators  

---

## ⚙️ Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn  
- FastAPI  
- Pydantic  

---

## 🔍 Approach

1. Data preprocessing and cleaning  
2. Exploratory Data Analysis (EDA) to identify key patterns  
3. Feature selection and transformation  
4. Model building using Random Forest Classifier  
5. Model evaluation and validation  
6. Deployment using FastAPI  

---

## 🤖 Model
- Algorithm: Random Forest Classifier  
- Performance: **90% prediction accuracy**  

---

## 🚀 Deployment
The model is deployed as a FastAPI application that:

- Accepts input data through API requests  
- Returns real-time scholarship eligibility predictions  
- Ensures input validation using Pydantic  

---

## 📈 Key Insights
- Academic performance and department are strong indicators of scholarship eligibility  
- Predictive modeling can significantly reduce manual evaluation effort  

---

## 💡 Business Impact
- Automates scholarship screening process  
- Improves consistency in decision-making  
- Reduces administrative workload  
- Enables data-driven evaluation
